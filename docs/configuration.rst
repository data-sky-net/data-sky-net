Configuration
-------------

Project settings are generally made while user answers wizard's questions in CLI. For advanced project customization you can manually change it's template which is based on `Cookiecutter <https://github.com/cookiecutter/cookiecutter>`_ Python tool. Read more in :ref:`Project Wizard templating <ds_project_deploy_wizard_templating_ref>`
